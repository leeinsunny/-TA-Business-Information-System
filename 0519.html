생성형 AI, 한눈에 이해하기

생성형 AI는 거대한 대규모 코퍼스인터넷 글·책·논문 등 방대한 텍스트 모음를 읽고 글쓰기 습관을 배웁니다.
그런 다음 확률 모델‘다음 단어가 나올 가능성’을 계산해 주는 수학 공식을 써서 가장 자연스러운 단어를 골라 새 문장·코드·그림을 만들어 냅니다.

🛠 학습은 이렇게 진행돼요

  1. Pre-training아무 글이나 읽으며 언어 감각을 익히는 단계 – ‘말하기 근육’ 키우기
  2. RLHF사람이 좋은 답·나쁜 답을 표시해 AI가 보상을 받도록 학습 – 예의 바른 답변 훈련
  3. Fine-tuning법률·의학 등 특정 분야 자료로 추가 공부 – 전문 지식 업그레이드

⚙️ 답변은 이렇게 만들어져요

여러분이 질문(프롬프트)을 던지면 AI는 Softmax단어마다 점수를 매겨 확률 분포를 만듦를 계산해 Top-k확률 상위 k개만 남기는 방법 또는 Top-p누적 확률이 p(예: 0.9) 이하가 되도록 자르는 방법으로 후보를 줄입니다.
마지막에 Temperature0이면 안전·사실 중심, 1이면 자유·창의 중심를 곱해 얼마나 ‘튀는’ 답을 낼지 결정합니다.

💡 어디에 쓸 수 있을까?

  • ✍🏻 보고서·이메일 초안 자동 작성
  • 🔍 긴 논문을 쉽고 짧게 요약
  • 🧑‍💻 코드 자동 완성·오류 설명
  • 🎨 게임 시나리오·이미지 스케치
💬 Tip | 정보성 글: Temperature 0.2 | Top-p 0.9
창작 스토리: Temperature 0.8 | Top-p 0.95

프롬프트 엔지니어링 첫 걸음

프롬프트(prompt)는 AI에게 보내는 “지시문”입니다.
질문이 구체적·명확할수록 답변 품질이 높아집니다.

⭐ 5가지 핵심 요소

  • 역할(Role) : “당신은 📉환경 컨설턴트입니다.” 왜?AI가 말투·전문 용어를 그 역할에 맞춰 바꿉니다.
  • 목표·형식(Task & Format) : “300단어 에세이를 제목·본문·결론 구조로 작성”
  • 제약조건(Constraint) : “숫자 데이터는 출처를 괄호로 표기”
  • 톤·대상(Audience) : “대학생이 이해하기 쉬운 어휘 사용”
  • 예시(Example) : 좋은 답변 샘플을 붙이면 AI가 유사한 스타일을 모방합니다.
ROLE → TASK → FORMAT → CONSTRAINT → Audience → EXAMPLE
💡 Iterative Prompting : 짧게 질문 → 결과 확인 → 조건 추가 → 다시 질문.
이렇게 2~3번 반복하면 원하는 품질에 도달하기 쉽습니다.

정보 검증 & 윤리

AI는 때때로 Hallucination사실처럼 보이지만 틀린 정보를 만들어내는 현상 을 일으켜, 그럴듯해 보이는 가짜 통계·날짜·인용문을 제시합니다.
학술·보고용으로 쓰려면 반드시 출처를 재확인해야 합니다.

🔍 검증 체크리스트

  • 👓 2차 검증 : Google Scholar · PubMed · 통계청 원문 대조
  • 🔗 URL 확인 : .edu, .gov 같은 신뢰성 높은 도메인 우선 사용
  • 🖋 인용·참고문헌 : 자동 생성 목록도 직접 클릭해 존재 여부 확인

⚖️ 윤리 가이드라인

  • 🪪 개인정보 입력 금지 : 이름·학번·주소 등 민감 정보는 프롬프트에 쓰지 않기
  • ©️ 저작권 : 생성 결과를 외부에 배포할 때는 “AI 작성·수정” 표기
  • 🤝 공정 사용 : 과제·보고서에 활용할 땐 본인 검토·수정 후 제출
💡 Tip : AI가 제시한 수치가 소수점 3자리·너무 깔끔 하면 의심해 보고 원본 논문을 찾아보세요.

1. Chat gpt

ChatGPT는 말을 걸면 바로 답을 주는 “대화형 인공지능”입니다.
과제 아이디어를 묻든, 파이썬 오류를 묻든 한 채팅창에서 이어서 질문 → 수정 → 재질문이 가능합니다.
콘텍스트 길이는 최대 32k 토큰토큰=단어·기호 단위. 32k는 한글 약 24,000자 정도까지 기억해 앞서 나눈 대화 맥락을 잊지 않습니다.

1) 버전 & 요금제

  • 🆓 무료 – GPT-3.5, 속도 빠르고 일상 Q&A에 충분
  • 💎 Plus (월 20$) – GPT-4, 논리 추론·코딩·분석 성능 ↑
  • 🔌 플러그인 및 GPT-4o – 웹 검색·PDF 읽기·코드 실행까지 확장

2) 장점 & 주의점

  • 다양한 언어 지원 → 다국어 번역·비교 학습에 유리
  • 빠른 프로토타입 → 5분 만에 보고서 초안·SQL 쿼리 완성
  • 최신 정보 한계 → 2023-04 이후 데이터는 “웹 검색” 플러그인 필요
  • Hallucination 가능 → 숫자·인용구는 2차 검증 필수
💡 팁 : “이 답변에 사용한 근거 링크를 3개 제공해 줘”라고 추가 질문하면, 검증에 도움이 되는 URL을 받을 수 있습니다.

실습 ① ChatGPT – 여름방학 7일 제주 여행 플래너

이번 실습에서는 ChatGPT를 활용해 여름방학 동안 7일간 제주도를 여행하는 일정표를 만들어 봅니다.
프롬프트를 조금씩 다듬어 갈수록 일정이 얼마나 현실적·맞춤형으로 바뀌는지 경험해 보세요!

▶ 실습 URL : chat.openai.com (로그인 후 진행)

기본 프롬프트

“7월 초, 대학생 2명이 7일간 제주도를 여행할 계획을 표로 짜 줘.”

🔄 단계별 프롬프트 개선

  1. v1 :
    “7월 초, 대학생 2명이 7일간 제주도를 여행할 계획을 표로 짜 줘.”
  2. v2 : 제약 추가
    “1인당 예산 60만 원, 렌터카 사용 조건으로 7일 제주 여행 일정을 표로 작성해 줘.”
  3. v3 : 형식 확장
    “1인당 예산 60만 원, 렌터카 사용. 각 날짜에 맛집·액티비티 링크 포함한 7일 제주 여행 일정을 표로 작성해 줘.”
  4. v4 : 조건 분기 추가
    “1인당 예산 60만 원, 렌터카 사용. 각 날짜에 맛집·액티비티 링크 포함하고, 우천 시 대안 일정까지 제시한 7일 제주 여행 플랜을 표로 작성해 줘.”

📊 ‘루브릭’이 뭐예요?

루브릭(Rubric) = AI의 응답결과를 평가하는 ‘평가 기준표’입니다.
미리 평가 항목과 점수 비율을 정해 두면, 결과를 공정하게 비교하고 피드백하기 쉽습니다.

항목 설명 배점
🚌 현실성 교통 시간·예산이 실제 가능한가? 40%
🎢 다양성 자연·맛집·체험이 골고루 존재하는가? 30%
🔗 정보 정확성 주소·링크가 실제 존재하는가? 20%
👁‍🗨 가독성 일정표가 한눈에 보기 쉬운가? 10%
💡 왜 평가할까?
• 자기 평가로 부족한 점을 발견하고 프롬프트를 개선할 수 있어요.
• 친구끼리 교차 평가를 하면 새로운 아이디어와 피드백을 얻을 수 있어요.

👉 실습이 끝나면 옆 친구와 일정표를 바꿔 점수를 매겨 보세요!

2. Claude

Claude 3 시리즈는 “길고 복잡한 문서 요약”에 특화된 AI입니다.
최상위 모델 Opus는 한 번에 100k 토큰한국어 약 7~8만 자, PDF 40쪽 분량을 기억해 논문·전자책·보고서를 통째로 넣어도 단락 요약·키워드 추출·질의응답을 바로 제공합니다.

2) 버전 · 요금제 (2025년 기준)

  • 🆓 Claude 3 Haiku – 20k 토큰, 무료 / 기본 Q&A용
  • 💎 Claude 3 Sonnet – 60k 토큰, 0.004 $ / 1k input
  • 💎 Claude 3 Opus – 100k 토큰, 0.008 $ / 1k input
  • 🔗 모든 버전 – Slack 봇·웹·API로 사용 가능

3) 장점 · 주의점

  • 장문 요약 특화 – 긴 PDF도 한 줄 “TL;DR”로 요약
  • 안전 필터 강화 – 편향·부적절 콘텐츠 자동 차단
  • 비용 효율 – GPT-4 대비 토큰 단가 약 4∼5배 저렴
  • 언어 선택 – 영어·한국어 우수, 희소 언어 지원은 제한
  • 실시간 웹 검색 부재 – 최신 뉴스는 직접 링크 제공 필요
💡 팁 : “표·그림만 뽑아 줘”처럼 세부 요청도 훌륭히 처리합니다.
팀 과제로는 Slack 채널에 Claude 봇을 초대해 요약-공유 채팅을 열어 보세요!

실습② Claude – 연근전 PDF 초압축 분석

위키백과 “연근전” 문서 PDF(약 2쪽 분량)를 업로드하고,
요약 → 조리 과정 → 영양 정보 → 퓨전 레시피까지 단계별로 받아봅니다.

🔗 준비 단계 ― PDF 내려받기 & 업로드

  1. ko.wikipedia.org ▶ 연근전 페이지 열기
  2. 오른쪽 사이드바 인쇄/내보내기 ▸ PDF로 다운로드 클릭 → PDF 저장
  3. Claude 웹 claude.ai 접속 ▶ New Chat ▶ 저장한 PDF 업로드

기본 프롬프트

“연근전 Wikipedia PDF를 3문장으로 요약해줘.”

🔄 단계별 개선 프롬프트

  1. v1 :
    “연근전 Wikipedia PDF를 3문장으로 요약해줘.”
  2. v2 조리 과정 :
    “연근전의 전통 조리 과정을 4단계로 간단히 설명해줘.”
  3. v3 영양 정보 :
    “연근전의 주요 영양 성분 3가지와 각 성분의 건강 효과를 한 줄로 설명해줘.”
  4. v4 퓨전 레시피 :
    “연근전을 활용한 현대식 퓨전 레시피 아이디어 3가지 제안해줘.”
👉 각 버전의 프롬프트를 복사→붙여넣기해 v1~v4 결과를 비교하며
Claude의 단계별 응답 변화를 직접 확인해 보세요!

3. Google Gemini

Google Gemini는 Google Cloud AI가 제공하는 차세대 멀티모달텍스트·이미지·음성 등 다양한 형태의 입력을 동시에 이해하고 처리하는 기술 LLM입니다.
텍스트·이미지·코드 입력을 모두 이해하며, 실시간 웹 인터페이스브라우저에서 바로 사용하는 사용자 화면를 통해 사실성 높은 답변을 제공합니다.

1) 버전 · 요금제 (2025)

  • 🆓 Bard 무료 – 웹 인터페이스에서 텍스트·이미지 대화 가능
  • ☁️ API다른 프로그램이 기능을 호출할 수 있도록 제공되는 프로그래밍 인터페이스 Standard – 1k 토큰단어·기호 등 텍스트의 최소 단위당 0.006 USD (텍스트)
  • 🚀 API Ultra – 1k 토큰당 0.12 USD (대용량·멀티모달 지원)

2) 장점 · 주의점

  • 웹 검색 통합 – 최신 뉴스·데이터를 바로 반영
  • 멀티모달 지원 – 이미지 설명·분석부터 코드 생성까지 가능
  • API 요금제 복잡 – Standard●Ultra 간 비용 차이 확인 필수
  • 접근성 제한 – 일부 국가·프로젝트에서만 사용 가능
  • 출력 길이 제한 – 텍스트 최대 128k 토큰, 이미지 캡션은 짧게 유지
💡 Tip
• “system: 당신은 데이터 분석가입니다” 역할 지시로 전문적 어조 설정
• “Show web sources” 요청 시, 출처 URL을 함께 출력
• 이미지 업로드 후 “Describe objects & sentiment” 입력 시 이미지 분석까지 수행

“홍대에서 요즘 뜨는 카페 5곳을 이모지 섞어 소개해 줘 — 각 카페 한글 이름·시그니처 메뉴·짧은 해시태그 포함(150자 이내)”

🔄 단계별 프롬프트

  1. V1 기본 :
    “홍대에서 요즘 뜨는 카페 5곳을 이모지 섞어 소개해 줘 — 각 카페 한글 이름·시그니처 메뉴·짧은 해시태그 포함(150자 이내)”

    “💗·🌿 이모지를 적극 사용해서 홍대 핫플 카페 5곳을 친구처럼 친근하게 소개해 줘 — 각 카페 이름·시그니처 메뉴·해시태그(150자 이내)”
  3. V3 플레이리스트 :
    “홍대 핫플 카페 5곳 소개 캡션에 어울리는 플레이리스트 곡 3곡도 함께 추천해 줘 — 각 곡 제목·아티스트 포함”

  • 150자 이내 글자 수·해시태그 형식이 정확한가?
  • 추천된 플레이리스트 곡이 실제 존재하는 곡인가?
